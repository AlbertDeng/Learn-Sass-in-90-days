## $susy預設版型設定

```
$susy: (
  flow: ltr、rtl,
  math: fluid、static,
  output: float、isolate,
  gutter-position: after,
  container: auto,
  container-position: center、left、right,
  columns: 4,
  gutters: .25,
  column-width: false 自訂,
  global-box-sizing: content-box,
  last-flow: to,
  debug: (
    image: hide,
    color: rgba(#66f, .25),
    output: background,
    toggle: top right,
  ),
  use-custom: (
    background-image: true,
    background-options: false,
    box-sizing: true,
    clearfix: false,
    rem: true,
  )
);
```
